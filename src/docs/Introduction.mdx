## Quick Start

### Installation

```bash
npm install @skai/ui
```

### Setup Tailwind CSS

Add the SKAI UI content path to your `tailwind.config.ts`:

```typescript
// tailwind.config.ts
export default {
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/@skai/ui/dist/**/*.{js,ts,jsx,tsx}",
  ],
  // ... rest of config
};
```

### Import Components

```tsx
import { Button, Card, Input } from "@skai/ui";

function MyComponent() {
  return (
    <Card>
      <Input placeholder="Enter amount" />
      <Button>Swap</Button>
    </Card>
  );
}
```

## Design Principles

### 1. Trading-First

Every component is designed with crypto/DeFi trading in mind. Colors, states, and patterns reflect trading workflows.

### 2. Accessible

Built on Radix UI primitives ensuring keyboard navigation, screen reader support, and ARIA compliance.

### 3. Composable

Small, focused components that combine to create complex interfaces.

### 4. Dark Mode Default

Designed for dark mode (trader preference) with light mode support.
